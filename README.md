# JavaSudokuSolver
 Java sudoku solver

## Install and use

### 1.
```
git clone https://github.com/eeetaF/JavaSudokuSolver.git
```

### 2.
Paste your test into `tests/temp_test.txt`

### 3.
Go to project's dir in your console (for example, `cd ~/Documents/GitHub/JavaSudokuSolver`)

### 4.
```
java Main.java
```
